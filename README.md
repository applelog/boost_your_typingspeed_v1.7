## Overview

**Boost Your Typing Speed v1.7** is a web-based typing practice program developed as part of the **KOICA Project Volunteer Group 3's ICT Capacity Building Project in Uganda**. This open-source tool empowers learners to develop touch typing skills through engaging, interactive challenges — from classrooms to community centers and beyond.

Whether you're starting to practice touch typing or want to improve your speed and accuracy, this program provides real-time feedback, performance analytics, and multiple game modes to keep learning engaging and effective.

---

## Key Features

- **Four Practice Modes**
  - Practice Mode: Master key positioning with guided letter sequences
  - Sentence Mode: Type realistic sentences and improve muscle memory
  - Test Mode: 60-second timed test with automatic WPM and accuracy calculation
  - Race Mode: Compete against an AI opponent to win the race

- **Real-Time Analytics**
  - WPM (Words Per Minute) measurement
  - Accuracy and mistake tracking
  - Level progression from Beginner to Expert
  - Performance history and stats

- **Responsive & Accessible**
  - Optimized for desktop, tablet, and mobile devices
  - High-contrast design for visibility
  - Keyboard-first interaction
  - No dependencies — pure HTML5, CSS3, and vanilla JavaScript

- **Bilingual Interface**
  - Complete English support
  - Full Korean (한국어) support
  - Bilingual patch notes

- **Offline-First**
  - Works completely offline
  - No internet or backend server required
  - Perfect for low-bandwidth environments

## How It Works

**Practice Mode**  
Type displayed letters to build muscle memory for proper key positioning.

**Sentence Mode**  
Complete full sentences with real-time error highlighting. Press Enter to advance to the next sentence.

**Test Mode**  
Type as much as possible in 60 seconds. The program calculates WPM, accuracy, and your typing level.

**Race Mode**  
Type a paragraph faster than the AI opponent. Reach 20 words ahead to win.
